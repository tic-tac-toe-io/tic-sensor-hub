# tic-sensor-hub

Data hub for time series data from TOE devices or TAC apps.


## Examples

- [data-upload-toe1-legacy](examples/data-upload-toe1-legacy) demonstrates uploading sensor data to SensorHub with TOE 1.0 data archive format.
